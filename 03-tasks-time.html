Stage 03

Tasks & time

Every project

Before you edit, everything runs through Asana. Find your task, open the project Juan sends, and track your time on it with TMetric. First project on this machine? Do the one-time setup at the bottom.

Each task

  1. Open your task in AsanaRead the brief in the description and note the due date.
  2. Open the project fileJuan attaches the .drp with everything loaded. Save it in your project's 01_Project/01_Resolve and open it — the proxies relink.
  3. Start the timer from the taskClick the green Start timer button on the Asana task.
  4. Set the project the first timeOn a project's first task, pick the project in TMetric so the time logs to the right place.
  5. Verify, then keep both openCheck task · tag · project are right in the browser. Keep TMetric + Asana open and stop/play the timer as you work. Download isn't timed — time runs on Selections, Editing and Adjustments.

Reference · the board

Every project is one board with three columns. Open one to see its tasks — each named prefix | name | version.

  • D |Download — pull the material. Not time-tracked; just mark it done so the stage is clear.
  • S |Selections — narrow down the footage.
  • E |Editing — the first cut (v1).
  • AD |Adjustments — revision rounds (v2, v3…).
  • D |Deliver — the client export.

One-time setup · once per machine

  • 1TMetric Desktop (Mac) — tracks idle time. In settings, set Detect idle time → Ask me with a 5-minute timeout. Step away that long and it asks what to do with the time — keep, discard, or mark as a break.
  • 2TMetric browser extension — install it, turn on the Asana integration (TMetric → Integrations → Asana), and refresh Asana. A green Start timer button appears on every task.

Remember

+Read the brief before you touch anything.
+Keep TMetric + Asana open in the browser the whole session.
–Don't trust the timer blindly — wrong task · tag · project means time logged in the wrong place.
